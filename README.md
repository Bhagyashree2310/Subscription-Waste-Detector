# Subscription Waste Detector

AI-powered dashboard to detect recurring subscriptions, monitor renewal reminders, and reduce unnecessary spending.

## Features
- Detects recurring subscriptions from expense data
- Tracks monthly subscription costs
- Shows renewal reminders
- AI-powered cost-saving suggestions
- Interactive dashboard with charts and analytics

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- Groq API

## Sample Subscriptions
- Netflix
- Spotify
- Amazon Prime
- ChatGPT
- Canva
- YouTube
- Google One

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
